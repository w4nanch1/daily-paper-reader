## 摘要
世界模型为长程LLM智能体提供了一种有原则的方法来赋予其预见能力：在执行之前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至降低下游决策的质量。在本文中，我们介绍了WorldEvolver，一种自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修改其部署时的上下文。WorldEvolver集成了三个模块：（i）情节记忆，通过基于检索的模拟利用真实行动转换；（ii）语义记忆，从预测-观察不匹配中提取持久的启发式规则；（iii）选择性预见，在将低置信度预测集成到智能体推理上下文之前对其进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个主干模型上实现了最高的预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修订提高了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

### 2. 方法论
- **核心思想**：将部署交互中产生的真实转换（情节记忆）和预测‑观察不匹配衍生的规则（语义记忆）作为可积累的外部证据，并引入置信度门控过滤低质量预测，使世界模型在推理时持续自我修正。
- **三个关键技术模块**：
    - **Episodic Memory（情节记忆）**：存储已执行的真实转换三元组 `(observation, action, next observation)`。对当前候选动作，通过**动作级Jaccard相似度**检索最相似的 `k_ME` 条历史，以原始文本形式注入世界模型提示。
    - **Semantic Memory（语义记忆）**：当预测与真实观察的**因子化状态元组**（通过映射函数 `g` 将文本转换为 `(subject, predicate, object)` 三元组）不一致时，使用**LLM批评器**提取持久性启发式规则（如“检查物体不会移动它”），每条规则附带证据分数 `e_i`，仅保留 `e_i > 0` 的规则，按小批量累积更新。
    - **Selective Foresight（选择性预见）**：基于生成预测的**几何平均token概率**计算置信度 `q_t = exp(ℓ_t)`，仅当 `q_t ≥ τ` 时才将预测传递给下游智能体，否则提供空信号（`∅`），实现不确定性下的“弃权”机制。
- **算法流程（Algorithm 1）**：
    1. 从冻结策略采样草稿动作 `a_t^{(0)}`。
    2. 检索情节记忆得到 `k_ME` 条最相似转换。
    3. 世界模型基于状态、动作及记忆生成预测 `ô_{t+1}` 与置信度 `q_t`。
    4. 选择性过滤：若 `q_t ≥ τ` 则 `F_t = ô_{t+1}`，否则 `F_t = ∅`。
    5. 智能体策略在条件 `(s_t, F_t)` 下采样最终执行动作 `a_t`。
    6. 若最终动作与草稿不同，重新查询世界模型获取对齐的预测。
    7. 执行动作，观察真实后续状态，更新情节记忆（追加新转换），并基于预测‑观察不匹配提取规则更新语义记忆。

### 7. 优点
- **免参数更新**：完全通过外部记忆修订实现自我进化，避免了灾难性遗忘和计算开销。
- **模块化设计**：情节记忆（利用）+ 语义记忆（探索）+ 选择性过滤（风险控制），分工明确，可独立替换或增强。
- **强鲁棒性**：置信度门控有效滤除不可靠预测，在弱骨干上尤其显著。
- **跨场景验证**：两个交互环境、两种智能体类型、多种骨干模型，显示方法具有较强的泛化能力。
- **理论连接**：明确区分情节与语义记忆（Tulving 理论），与认知科学相关概念对应，增强了方法的可解释性。

### 8. 不足与局限
- **评估范围有限**：仅测试了 ALFWorld 和 ScienceWorld 两个文本环境，未涵盖网页导航、代码生成、机器人、多模态交互等更复杂领域。

- **语义记忆提取开销**：每次不匹配都需要 LLM 批评器生成规则，可能引入额外延迟和不一致；规则质量依赖批评器本身能力。
