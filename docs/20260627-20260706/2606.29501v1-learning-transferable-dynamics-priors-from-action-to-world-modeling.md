## 摘要
我们研究动作条件下的世界建模作为学习机器人可迁移动态先验的可扩展方式。通过预训练一个模型来预测动作如何驱动视觉场景演变，所得的世界模型捕获了超越外观层面视频生成的可重用交互动态。具体而言，我们在带有真实动作标注的大规模机器人操作数据上预训练了一个多视角交互基础扩散世界模型 A2World。我们从两个互补的角度验证学习到的动态先验。首先，我们将 A2World 适配为任务或场景专属的真实世界模拟器 A2World-sim，其长程 rollout 通过用世界模型 rollout 替代真实机器人 rollout，支持基于模拟器的策略评估和可扩展的假设分析。其次，从相同的预训练权重出发，我们将 A2World 适配为视频-动作联合预测模型 A2World-policy，该模型在视觉和指令条件下预测动作。在仿真基准和真实机器人环境中的实验表明，动作条件下的世界模型预训练产生了可迁移的动态先验，有益于以模拟器为中心和以策略为中心的机器人学习。

## Abstract
We study action-conditioned world modeling as a scalable way to learn transferable dynamics priors for robot learning. By pretraining a model to predict how actions drive visual scene evolution, the resulting world model captures reusable interaction dynamics beyond appearance-level video generation. Concretely, we pretrain a multi-view interactive base diffusion world model, A2World, on large-scale robot manipulation data with real action annotations. We validate the learned dynamics priors from two complementary perspectives. First, we adapt A2World into a task- or scene-specialized real-world simulator, A2World-sim, whose long-horizon rollouts support simulator-based policy evaluation and scalable what-if analysis by replacing real-robot rollouts with world model rollouts. Second, starting from the same pretrained weights, we adapt A2World into a video-action joint prediction model, A2World-policy, that predicts actions under visual and instruction conditioning. Experiments across simulation benchmarks and real-robot settings demonstrate that action-conditioned world model pretraining yields transferable dynamics priors that benefit both simulator-centric and policy-centric robot learning.

## 2. 论文提出的方法论

### 核心思想
- 在大规模真实机器人操作数据上预训练一个**多视角扩散世界模型**（A2World），输入初始帧和未来动作块，预测未来多视角视频。将预训练权重视为可迁移的动态先验，然后通过最小修改适配为两种变体：
  - **A2World-sim**：添加历史条件，支持长程自回归 rollout 作为真实世界模拟器。
  - **A2World-policy**：MoE-like 架构，共享视频和动作的自注意力，预测未来视频和动作，实现指令条件下的策略。

### 关键技术细节

#### A2World（基础世界模型）
- **动作条件**：动作块通过MLP编码为嵌入，添加到扩散时间步嵌入，注入每个DiT块的AdaLN。
- **多视角生成**：将多个视角的潜在序列沿时间维度拼接，并引入可学习视角嵌入和**跨视角注意力**模块，增强一致性和细节。
- **预训练目标**：EDM去噪得分匹配，训练时禁用交叉注意力条件（设为0），仅依靠动作调制。

#### A2World-sim（长程模拟器）
- **历史注入**：通过**姿态引导历史采样**（Pose-guided History Sampling，算法1和2）从历史帧中选取紧凑的关键状态，然后通过交叉注意力和自注意力记忆注入DiT。
- **自回归生成**：使用**自强制（Self-forcing）**训练策略，用模型自身生成的帧作为条件，增强长程稳定性。

#### A2World-policy（视频-动作联合预测模型）
- **MoE-like设计**：视频和动作令牌**共享自注意力模块**，但各保留独立的AdaLN和MLP分支。动作令牌通过线性头预测。
- **联合扩散**：对视频和动作潜在分别加噪，使用共享基噪声层级但按模态缩放（$m_v=\sqrt{6}, m_a=0.5$），加权联合去噪损失。
- **推理**：使用模态级无分类器引导，可独立控制视频和动作的引导权重。

## 6. 论文的主要结论与发现
1. **动作条件世界模型预训练产生可迁移的动态先验**，远优于文本条件或通用视频生成预训练（如 Cosmos-Predict2）。
2. **该先验可有效适配为两种互补用途**：A2World-sim 作为真实世界模拟器（与真实成功率强相关，R²=0.930），A2World-policy 作为强策略（LIBERO 98.6%，真实机器人平均 83.7%）。

4. **历史注入**中的姿态引导采样优于滑动窗口或无历史；**自强制训练**对长程稳定性关键。
5. **视频预测和动作生成存在正耦合**：联合训练优于冻结视频分支，视频一致性改善伴随动作质量提升。
