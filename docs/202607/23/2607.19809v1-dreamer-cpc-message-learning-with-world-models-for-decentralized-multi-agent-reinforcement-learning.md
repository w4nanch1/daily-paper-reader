## 摘要
在多智能体强化学习（MARL）中，智能体间的通信能有效提升部分可观测条件下的性能。基于表示学习的方法使去中心化智能体能够学习基于自身观测的消息，但这些方法仅依赖当前观测，无法传递随时间累积的信息。我们提出Dreamer-CPC，一种将基于集体预测编码（CPC）的消息学习集成到DreamerV3世界模型中的去中心化基于模型的MARL方法。每个智能体独立维护一个世界模型和一个消息模块，并从反映过去观测与动作历史的世界模型潜在状态中推断并交换消息。我们在两个环境中评估了Dreamer-CPC：Observer，一个非合作信息共享任务；以及CatchApple，一个新引入的任务，其中任务相关观测暂时缺失。在两个环境中，Dreamer-CPC均优于IPPO-CPC（一种现有基于CPC的从当前观测生成消息的方法）以及无通信基线。特别是在CatchApple中，Dreamer-CPC获得的回合回报是IPPO-CPC的4到5倍，展示了在其他方法因观测缺失而失败的情况下实现有效协调的能力。这些结果表明，当仅凭当前观测不足时，基于世界模型潜在动力学的通信能够支持去中心化决策。

## Abstract
In multi-agent reinforcement learning (MARL), inter-agent communication is effective for improving performance under partial observability. Representation learning-based approaches enable decentralized agents to learn messages grounded in their own observations, but they rely only on current observations and cannot convey information accumulated over time. We propose Dreamer-CPC, a decentralized model-based MARL method that integrates message learning based on Collective Predictive Coding (CPC) into the world model of DreamerV3. Each agent independently maintains a world model and a message module, and infers and exchanges messages from the latent states of the world model that reflect the history of past observations and actions. We evaluated Dreamer-CPC in two environments: Observer, a non-cooperative information-sharing task, and CatchApple, a newly introduced task in which task-relevant observations are temporarily missing. In both environments, Dreamer-CPC outperformed IPPO-CPC, an existing CPC-based method that generates messages from current observations, as well as no-communication baselines. In particular, in CatchApple, Dreamer-CPC achieved 4 to 5 times the episode return of IPPO-CPC, demonstrating effective coordination where other methods fail due to missing observations. These results suggest that communication grounded in the latent dynamics of world models can support decentralized decision-making when current observations alone are insufficient.

## 1. 论文的核心问题与整体含义

在去中心化多智能体强化学习（MARL）中，智能体在部分可观测环境下通常需要借助智能体间通信来获取任务相关信息。现有基于表示学习的通信方法（如 MARL-CPC、IPPO-CPC）虽然能让智能体学习基于自身观测的消息，但它们仅利用当前时刻的观测生成消息，无法传递随时间累积的观察历史或预测未来状态所需的信息。当任务相关观测暂时缺失时（如物体被遮挡或需要提前预判），仅靠当前观测的通信方式会失效。本文提出 Dreamer-CPC，将集体预测编码（CPC）的消息学习与 DreamerV3 的世界模型（基于 RSSM 的循环状态空间模型）相结合，使智能体能够从自身历史观测与动作积累的潜在动力学状态中推断消息，从而在部分可观测且信息临时缺失的场景下支持有效的去中心化协调。

## 2. 方法论

- **核心思想**：每个智能体独立维护一个基于 RSSM 的世界模型和一个循环消息模块。世界模型（DreamerV3 风格）从观测、动作、来自其他智能体的消息中学习潜在状态表示；消息模块从世界模型的潜在状态（历史上下文）中推断消息，并与其他智能体交换。消息用于改善世界模型的预测（观测、奖励、延续标志）以及策略学习。
- **关键技术细节**：
  - **世界模型**：每个智能体 k 的 RSSM 包括确定性循环状态 h^k_t、随机潜在状态 z^k_t；先验 ẑ^k_t 仅从 h^k_t 采样，后验 z^k_t 从 h^k_t 和观测 x^k_t 推断；解码器根据 h^k_t、z^k_t 和当前收到的消息预测观测、奖励和结束标志。
  - **消息模块**：包含消息循环状态 ξ^k_t，通过 GRU 从上一时刻所有智能体的消息更新；消息先验 m̂^k_t 仅从 ξ^k_t 采样；消息后验 m^k_t 从 ξ^k_t、h^k_t、z^k_t 推断。消息在训练和想象 rollout 中交换。
  - **学习目标**：联合优化世界模型和消息模块，损失函数由预测损失、KL 散度（经自由位和平衡损失处理）组成，具体包括反映重建精度（L_pred）、动力学一致性（L_dyn、L_rep）、消息先验/后验一致性（L_m-dyn、L_m-rep）的项，权重分别为 β_pred=1, β_dyn=1, β_rep=0.1, β_m-dyn=1e-3, β_m-rep=1e-4。其他智能体的消息使用 stop gradient 以实现完全去中心化学习。

- **算法流程**（文字说明）：
  - 训练世界模型与消息模块时，从回放缓冲区采样轨迹，按时间步依次更新每个智能体的消息循环状态、推断消息、预测观测等，计算损失并更新参数（各智能体独立）。

## 3. 实验设计

- **环境与场景**：
  - **Observer**：非合作信息共享任务。Agent1 静止并知晓奖励位置，但无动作；Agent2 可移动，需通过通信从 Agent1 获取奖励位置信息以挖取奖励。全局状态可通过所有智能体观测完全重构。
  - **CatchApple**：新引入的任务，模拟物体掉落和抓取场景。Agent1 静止观测物体轨迹，Agent2 控制抓取器，但 Agent2 无法直接观测物体未来着陆列（物体在遮挡行被隐藏标志替代），必须依靠 Agent1 发送的消息提前预判并移动。部分时间步观测缺失，无法仅靠当前时刻聚合观测重构完整状态。

## 6. 主要结论与发现

- Dreamer-CPC 在 Observer 和 CatchApple 中均显著优于 IPPO-CPC 和无通信基线，表明基于世界模型历史潜在状态的通信比基于当前观测的通信更有效，尤其在观测临时缺失时。

## 7. 优点

- **方法创新**：首次将 CPC 消息学习集成到 DreamerV3 世界模型的潜在动力学中，使消息蕴含历史上下文而非仅当前观测，适用于部分可观测且信息时变缺失的场景。

- **验证环境设计**：CatchApple 环境巧妙体现了“当前观测不足”的挑战，能有效区分基于历史状态通信与基于当前观测通信的差异。
