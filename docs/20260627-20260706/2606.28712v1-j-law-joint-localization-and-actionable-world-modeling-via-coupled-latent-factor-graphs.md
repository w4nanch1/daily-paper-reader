## 摘要
经典SLAM估计度量位姿和几何地图，但无法为规划提供可操作的预测模型。动作条件世界模型学习用于规划的紧凑潜在动力学，但忽略全局度量一致性，并在开环展开下累积漂移。我们认为这是同一估计问题的两种视角，并在本文中提出J-LAW（联合定位与可操作世界建模）：一个耦合因子图，联合优化度量物体位姿、潜在世界状态和潜在地标嵌入。桥梁是一个位姿条件的潜在编码器和一个学习到的位姿-潜在耦合因子，使得更好的定位改善世界模型，反之亦然。我们将观测、动作条件预测、度量里程计、位姿-潜在耦合、潜在闭环和潜在地标观测视为单个MAP目标中的概率因子。在PushT和WildGS上的真实数据实验表明，相对于开环展开，耦合图校正显著降低了潜在预测RMSE和端点漂移，而潜在闭环改善了全局轨迹一致性。J-LAW产生一个同时是度量（位姿）和可操作（用于规划的潜在地标）的地图。

## Abstract
Classical SLAM estimates metric poses and a geometric map but produces no actionable predictive model for planning. Action-conditioned world models learn compact latent dynamics for planning but ignore global metric consistency and accumulate drift under open-loop rollout. We argue these are two views of the same estimation problem and propose J-LAW (Joint Localization and Actionable World Modeling) in this letter: a coupled factor graph that jointly optimizes metric object poses, latent world states, and latent landmark embeddings. The bridge is a pose-conditioned latent encoder and a learned pose--latent coupling factor, so that better localization improves the world model and vice versa. We cast observation, action-conditioned prediction, metric odometry, pose--latent coupling, latent loop closure, and latent landmark observation as probabilistic factors in a single MAP objective. Real-data experiments on PushT and WildGS show that coupled graph correction substantially reduces latent prediction RMSE and endpoint drift relative to open-loop rollout, while latent loop closure improves global trajectory consistency. J-LAW yields a map that is simultaneously metric (poses) and actionable (latent landmarks for planning).

  - **动作条件世界模型**（如JEPA，联合嵌入预测架构）学习紧凑潜在动力学用于规划，但**忽略全局度量一致性**，开环展开时漂移累积，且没有闭环修正机制。
- **核心问题**：两类方法本质上是同一估计问题的两个互补视角，但现有研究未能有效统一。
- **整体意义**：本文提出 **J-LAW**，首次将度量定位与可操作潜在世界建模在**耦合因子图**中联合优化，同时输出**度量地图（位姿）** 和**可操作地图（潜在地标，可直接用于规划）**，实现双向增益。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程
- **核心思想**：构建一个**耦合因子图**，联合估计度量位姿 \(X = \{x_t\}\)、潜在世界状态 \(Z = \{z_t\}\)、潜在地标嵌入 \(M = \{m_k\}\)。关键桥接是**位姿条件潜在编码器** \(z_t = E_\theta(o_t, x_t)\)，使潜在表示同时依赖观测和度量位姿。
- **关键技术细节**：
  - **联合最大后验（MAP）目标**：最小化包含六个因子项的损失函数：
    1. **潜在观测因子** \(\Phi_{obs}\)：锚定潜在状态到位姿条件编码。
    2. **动作条件潜在预测因子** \(\Phi_{pred}\)：JEPA风格的预测一致性。
    3. **度量运动模型因子** \(\Phi_{mot}\)：标准SLAM里程计因子（SE(3)）。
    4. **位姿-潜在耦合因子** \(\Phi_{couple}\)：学习解码器 \(h_\theta\) 从位姿预测潜在状态，形成双向约束。
    5. **潜在闭环因子** \(\Phi_{loop}\)：访问相同位置时，潜在状态趋于一致（含鲁棒核函数）。
    6. **潜在地标观测因子** \(\Phi_{lmk}\)：关联潜在状态与地标。
  - **交替块坐标下降（Alternating Block Coordinate Descent）**：
    - 直接联合优化易产生退化解（位姿和潜在相互靠近而不改进）。
    - 将状态分为**位姿块（\(X\)）** 和**潜在块（\(Z, M\)）**，交替优化其中一个块，固定另一个块。每块用L-BFGS + 强Wolfe线搜索求解，迭代K=5–8轮。
    - 位姿块固定潜在，耦合因子成为位姿的正则化项；潜在块固定位姿，耦合因子成为潜在的正则化项。此举避免退化，同时双向传递信息。
  - **训练与推理三阶段**：
    1. **预训练**：标准JEPA损失训练编码器 \(E_\theta\) 和预测器 \(P_\theta\)（忽略位姿）。
    2. **位姿接地**：在已知位姿数据上训练耦合解码器 \(h_\theta\)，最小化 \(\|z_t - h_\theta(x_t)\|^2\)。
    3. **交替推理**：冻结所有网络，运行交替块坐标下降求解耦合因子图。

## 3. 实验设计：数据集、基准、对比方法
- **数据集**：

- **基准（Baseline）**：

  - **仅位姿（Pose-only）**：只优化位姿，潜在状态由 \(z_t = h_\theta(x_t)\) 完全决定。

  - **J-LAW交替（Alternating）** 及其变体：固定耦合、自适应耦合（IRLS权重）。
- **对比维度**：

## 6. 论文的主要结论与发现
- **漂移显著减少**：因子图校正相对于开环展开，潜在RMSE降低**71–94%**（WildGS：0.3050 → 0.0873；终点误差2.5226 → 0.9511）。
- **闭环改善全局一致性**：真闭环将闭环误差最多降低98%，但以轻微点状RMSE增加为代价（一致性-精度权衡）。
- **交替优化优于联合优化**：J-LAW交替将潜在RMSE降至0.1383，优于联合L-BFGS（0.1664，甚至比潜在基线更差）和潜在基线（0.1595）。联合L-BFGS出现退化。
- **长时漂移稳定性**：J-LAW交替的端点漂移从H=48到512仅增长11%，而仅位姿增长38%，在H=512时J-LAW优于仅位姿（2.33 vs 3.11）。
- **自适应耦合改善位姿估计**：IRLS自适应权重使位姿RMSE降低29–42%，但潜在RMSE未改善（表明耦合解码器质量是瓶颈）。
